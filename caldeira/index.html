Modelo da Planta
ΔL(s), ΔP(s) = [ LF(s) LQ(s); PF(s) PQ(s) ] · [ ΔF(s); ΔQ(s) ] + [ LV(s); PV(s) ] · ΔV(s)
LF(s) = KF / (s(τF s + 1))
LQ(s) = KQ(τQz s − 1) / (s(τQp s + 1))
LV(s) = KV(τVz s − 1) / (s(τVp s + 1))
PF(s) = PQ(s) = PV(s) = K′i / s
